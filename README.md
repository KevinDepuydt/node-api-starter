# Node API Starter

Simple Node, Express app using JWT with sample model, routes and controller for users

## Scripts

### Start in production
```
npm start
```

### Start in development
```
npm run dev
```

### Build
transpile ES6 code to JS code
```
npm run build
```

### Clean
Clean dist directory that contain transpilled code
```
npm run clean
```

### Test
```
npm test
```
